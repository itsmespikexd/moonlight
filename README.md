# 🌙 Moonlight

**Moonlight** is a modern weather web app that lets you explore real-time weather conditions anywhere in the world — with a beautiful interface, interactive map, and rich atmospheric details.

---

## ✨ Features

🔍 **Search Any Location**  
Quickly find weather data for any city, country, or place worldwide.

🗺️ **Live Map Integration**  
See exactly where you're searching with an interactive map view.

🌡️ **Detailed Weather Metrics**  
Get more than just temperature:
- 🌡️ Temperature
- 🤔 Feels Like
- ☁️ Cloud Coverage
- 🌧️ Precipitation
- 👁️ Visibility
- 📊 Pressure
- And more...

---

## 🌌 Why Moonlight?

Most weather apps are cluttered or overwhelming.  
**Moonlight** focuses on simplicity, clarity, and a smooth user experience, which gives you exactly what you need.
